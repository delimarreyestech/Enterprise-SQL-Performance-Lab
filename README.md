# Enterprise SQL Performance Lab

A professional SQL Server performance engineering lab focused on realistic enterprise scenarios, query optimization, indexing strategies, execution plans, parameter sniffing, blocking, deadlocks and measurable performance improvements.

---

## Business Context

A multinational retail company is experiencing performance issues in its transactional SQL Server database.

Users report slow response times when searching orders, reviewing customer history and generating operational reports.

This project simulates that environment and documents the process of diagnosing, measuring and optimizing SQL Server workloads.

---

## Project Objectives

- Build a realistic SQL Server transactional database.
- Generate production-style datasets.
- Create slow queries and performance bottlenecks.
- Analyze execution plans.
- Design and test indexes.
- Demonstrate parameter sniffing.
- Simulate blocking and deadlocks.
- Measure before vs after improvements.
- Document findings using professional engineering standards.
